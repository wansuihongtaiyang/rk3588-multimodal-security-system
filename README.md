# 基于 RK3588 的多模态边缘智能安防巡检系统

本项目面向库房、实验室、设备间等无人值守重点区域，设计并实现了一套基于 RK3588 的多模态边缘智能安防巡检系统。系统融合摄像头、光照传感器、运动传感器、语音控制、人脸识别、目标检测、图像语义理解、风险决策和网页远程展示，实现从异常感知到事件归档的完整闭环。

## 主要功能

- 多模态感知：摄像头、BH1750 光照传感器、MPU6050 运动传感器
- 语音控制：支持开始检测、查看状态、抓拍、开启/关闭告警、夜间模式等命令
- 目标检测：基于 YOLOv5 RKNN 完成本地目标检测
- 人脸识别：基于 FaceAuth / SFace 完成人脸白名单判断
- AI 分析：结合 Qwen2-VL、DeepSeek / fallback 报告模块生成事件说明
- 加权风险决策：综合光照、运动、目标和身份信息输出风险等级
- 远程交互：支持 8080 异常事件页面、8090 AI 报告页面、9090 手机控制中心
- 证据归档：保存原图、检测图、传感器状态、人脸结果、AI 报告和安全决策文件
